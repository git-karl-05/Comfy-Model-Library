**Project status: Early development. Features, architecture, and APIs may change.**

Planned Features
    - Mobile-First Interface
    - Responsive layouts for phones, tablets, and desktops
    - Large touch-friendly controls
    - Interfaces that avoid drag-and-drop where possible
    - Fast access to commonly used actions

Local Gallery
    - Browse ComfyUI output folders
    - View images in responsive grids
    - Inspect generation metadata
    - Search prompts and generation parameters
    - Filter by date, model, workflow, sampler, dimensions, or folder
    - Mark images as favorites
    - Add ratings, tags, or collections

Workflow Library
    - Save and organize reusable workflows
    - Display workflow thumbnails and descriptions
    - Add tags, notes, and categories
    - Identify models and custom nodes required by a workflow
    - Create simplified launch forms for frequently used workflows
    - Preserve workflow versions
    - Associate generated images with the workflows that created them
    - Model and Asset Management
